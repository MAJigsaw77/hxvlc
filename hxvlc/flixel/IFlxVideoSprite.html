interface IFlxVideoSprite

package hxvlc.flixel

import hxvlc.flixel.FlxVideoSprite

extended by FlxVideoSprite

Interface representing a video sprite object in Flixel.

Variables

autoPause:Bool

Indicates whether the video should automatically pause when focus is lost.

Must be set before loading a video.

read onlybitmap:Video

The video bitmap object.

Methods

load(location:Location, ?options:Array<String>):Bool

Loads a video from a specified location.

Parameters:

location

The path, URL, file descriptor ID, or bitstream input of the media.

options

Additional options for LibVLC Media.

Returns:

true if the video loads successfully, false otherwise.

loadFromSubItem(index:Int, ?options:Array<String>):Bool

Loads a media subitem from the current media's subitems list at the specified index.

Parameters:

index

The index of the subitem to load.

options

Additional options to configure the loaded subitem.

Returns:

true if the subitem was loaded successfully, false otherwise.

parseStop():Void

Stops the parsing of the current media item.

parseWithOptions(parse_flag:Int, timeout:Int):Bool

Parses the current media item with the specified options.

Parameters:

parse_flag

The parsing option.

timeout

The timeout duration in milliseconds.

Returns:

true if parsing succeeds, false otherwise.

pause():Void

Pauses video playback.

play():Bool

Starts video playback.

Returns:

true if playback started successfully, false otherwise.

resume():Void

Resumes playback of a paused video.

stop():Void

Stops video playback.

togglePaused():Void

Toggles between play and pause states of the video.
